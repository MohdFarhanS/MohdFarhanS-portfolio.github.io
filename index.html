Jun 2026 Personal Project

NewsPortal

Indonesian news portals are often static and editor-heavy. NewsPortal is a production CMS with a structured editorial workflow, real-time trending, and full multi-role access, built on Next.js 15 and TypeScript end to end.

  • Multi-role editorial workflow: 4 user tiers (USER, JOURNALIST, EDITOR, ADMIN) with DRAFT → REVIEW → PUBLISHED pipeline enforced via Next.js middleware and Auth.js v5 JWT
  • Real-time trending engine, view tracking per IP with 24-hour deduplication and 7-day rolling window; full-text search with category, tag, and date filters via Prisma
  • Normalized PostgreSQL schema across 9 models; Upstash Redis rate limiting, Cloudinary image optimization, Resend password reset, deployed on Vercel
